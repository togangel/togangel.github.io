Instalación

Abre el programa de instalación haciendo doble clic en su icono.

Si el idioma de tu Windows no es compatible con el instalador, selecciona un idioma alternativo que desees utilizar (tanto para la instalación como posteriormente al usar WinSCP).

En la pantalla del Acuerdo de licencia, haz clic en "Aceptar" después de revisar la licencia.

Se te pedirá que selecciones un tipo de configuración. Para una configuración básica, elige "Instalación típica".

Luego se te pedirá la configuración inicial del usuario. Esto te permitirá seleccionar el estilo de interfaz de usuario. Si has utilizado (y te ha gustado) administradores de archivos como Total Commander, Altap Salamander o Norton Commander anteriormente, mantén la interfaz Commander predeterminada. De lo contrario, es probable que estés más familiarizado con la interfaz Explorer.

Después de seleccionar la interfaz, se abrirá la pantalla "Listo para instalar". En esta pantalla, puedes revisar las opciones de instalación que has seleccionado. Haz clic en "Instalar" para iniciar la instalación.

Se llevará a cabo un breve proceso de instalación. Es posible que debas reiniciar el Explorador de archivos de Windows o tu computadora. Si eliges no reiniciar, es posible que algunas funciones de WinSCP no estén disponibles hasta que lo hagas..

¡Listo! Ahora ya podrás hacer uso de la aplicación para realizar las descargas. (Cada vez que necesites descargar un archivo, se solicitará la contraseña del FTP para poder descargarlo.)
